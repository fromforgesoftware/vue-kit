# Color Swatch Picker

A grid of selectable colour swatches built on Reka UI's listbox primitive. Use this when the choice is constrained to a predefined palette — theme tokens, brand colours, label colours, status colours.

## When to use

- Predefined palettes (theme tokens, brand colours).
- Quick selection in a popover where a full picker would be overkill.
- Multi-select chips for tag colour assignment.

## When not to use

- For arbitrary colour input — use [Color Picker](?path=/docs/color-color-picker--docs).
- For displaying a single colour — use [Color Swatch](?path=/docs/color-color-swatch--docs).

## Composition

`ColorSwatchPickerRoot` provides arrow-key navigation, selection state, and the `aria-label`. Each swatch is a `ColorSwatchPickerItem` containing:

- `ColorSwatchPickerItemSwatch` — the visible colour chip.
- `ColorSwatchPickerItemIndicator` — a check icon shown when selected.

## Multi-select

<Canvas of={ColorSwatchPickerStories.Multiple} />

Pass `multiple` and bind to a `string[]`. Each click toggles a value.

## Layouts

<Canvas of={ColorSwatchPickerStories.GridLayout} />

`flow` (default) wraps swatches naturally; `grid` enforces a 6-column grid for tightly packed palettes.

## Accessibility (WCAG 2.2 AA)

- **Group name** (SC 4.1.2): the root accepts `ariaLabel`. Default is `"Color palette"` — override with something specific (e.g. `"Label colour"`).
- **Swatch name**: each item carries an auto-derived `aria-label` from the colour value (e.g. `"red"`).
- **Selection state** (SC 4.1.2): selected items have `aria-selected="true"` and `data-state="checked"`. The check indicator is purely visual; the selection state is announced via the `aria-selected` attribute.
- **Target size** (SC 2.5.8): every size meets ≥ 24 × 24 px.
- **Focus visible** (SC 2.4.7): focus shows a ring with offset against the background.

## Responsive behaviour

<Canvas of={ColorSwatchPickerStories.InteractiveResponsive} />

The default `flow` layout wraps cleanly on narrow viewports. Use `grid` only when you control the parent width.
