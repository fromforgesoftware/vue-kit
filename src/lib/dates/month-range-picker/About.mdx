# MonthRangePicker

Composite month-range picker — button trigger that opens a popover with a 4 × 3 grid of months for picking a contiguous range. Built on Reka UI's `MonthRangePicker` primitives.

## When to use

- Reports requiring a contiguous month range (year-to-date, last 6 months).
- Filters where granularity below a month would be noise.
- Inline scheduling layouts via the `inline` variant.

## When not to use

- Single-month selection — use [MonthPicker](?path=/docs/dates-monthpicker--docs).
- Day-level ranges — use [DateRangePicker](?path=/docs/dates-daterangepicker--docs).
- Year-only ranges — use [YearRangePicker](?path=/docs/dates-yearrangepicker--docs).

## Accessibility (WCAG 2.2 AA)

- **2.4.6 Labels**: pair every picker with `<Label for="id">`.
- **3.3.1 Error identification**: `error` + `describedBy` over colour alone.
- **2.4.7 Focus visible**: 2 px ring on trigger, nav buttons, and cells.
- **2.5.8 Target size**: trigger and month cells meet ≥ 24 × 24 px at every size.
- **4.1.2 Name & state**: range start / end and highlighted preview cells expose Reka data attributes.
