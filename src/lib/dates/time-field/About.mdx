# TimeField

Segmented time input — hour, minute (and optional second + AM/PM) segments are independently focusable. Built on Reka UI's `TimeField` primitive.

## When to use

- Fast keyboard time entry inside a form (start/end time, duration).
- Free-typing a known time — segments accept digits and snap on overflow.

## When not to use

- Date entry — use [DateField](?path=/docs/dates-datefield--docs).
- Time-range entry — use [TimeRangeField](?path=/docs/dates-timerangefield--docs).
- Combined date + time entry — use [DateField](?path=/docs/dates-datefield--docs) with `granularity="minute"`.

## Anatomy

<Canvas of={TimeFieldStories.Default} />

`TimeFieldRoot` wraps hour and minute segments separated by literals (`:`). When `hourCycle={12}` an additional AM/PM toggle segment appears.

## Accessibility (WCAG 2.2 AA)

- **2.4.6 Labels**: pair every field with `<Label>`.
- **3.3.1 Error identification**: pass `error` and `describedBy`.
- **2.4.7 Focus visible**: focused segments highlight with `bg-accent`; the wrapper renders a 2 px ring.
- **2.5.8 Target size**: every size's wrapper meets ≥ 24 × 24 px.
- **4.1.2 Name & state**: each segment exposes `role="spinbutton"` with min / max bounds.
