# YearRangePicker

Composite year-range picker — button trigger that opens a popover with a paged grid of years for selecting a contiguous range. Built on Reka UI's `YearRangePicker` primitives.

## When to use

- Multi-year reports and KPI dashboards.
- Filters where the smallest meaningful unit is a calendar year.
- Inline scheduling layouts via the `inline` variant.

## When not to use

- Single-year selection — use [YearPicker](?path=/docs/dates-yearpicker--docs).
- Month-level ranges — use [MonthRangePicker](?path=/docs/dates-monthrangepicker--docs).

## Anatomy

<Canvas of={YearRangePickerStories.Default} />

`YearRangePickerRoot` wraps a header with page navigation and a 4-column grid of years. `yearsPerPage` controls how many years are visible.

## Accessibility (WCAG 2.2 AA)

- **2.4.6 Labels**: pair every picker with `<Label for="id">`.
- **3.3.1 Error identification**: `error` + `describedBy`.
- **2.4.7 Focus visible**: 2 px ring on trigger, nav buttons, cells.
- **2.5.8 Target size**: trigger and year cells meet ≥ 24 × 24 px.
- **4.1.2 Name & state**: range start / end and highlighted preview cells expose Reka data attributes.
