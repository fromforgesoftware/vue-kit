# AspectRatio

Locks a child element to a fixed aspect ratio regardless of its width. Wraps Reka UI's `AspectRatio` primitive — useful for media (images, embeds, video), card thumbnails, and chart placeholders.

## When to use

- Image / video tiles in a responsive grid where every cell should be the same shape.
- Reserving space for an asset that hasn't loaded yet (avoids layout shift).
- Keeping chart or map placeholders at a known shape.

## When not to use

- For elements whose height should follow their content (text blocks, lists).
- For tiny inline elements — use Tailwind utilities (`aspect-square`) instead.

## Anatomy

<Canvas of={AspectRatioStories.Default} />

## Accessibility (WCAG 2.2 AA)

- **1.1.1 Non-text Content**: AspectRatio is a layout helper — accessibility is delegated to the child. Always pass meaningful `alt` to wrapped images and proper labels to embeds.
- **1.4.10 Reflow**: percentage-based padding ensures the child reflows smoothly without horizontal scroll.

## Responsive behaviour

<Canvas of={AspectRatioStories.InteractiveResponsive} />

The wrapper inherits its width from its parent and recomputes height from the ratio. The aspect ratio is preserved at every breakpoint.
