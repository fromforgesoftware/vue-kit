# StatTrend

Compact badge that shows the **direction** and **magnitude** of a change. Use next to a stat value in dashboards, summary cards, and analytics tables.

## When to use

- Period-over-period comparison ("+12.5 % vs last week").
- Delta on a KPI in a dashboard card.
- Inline trend indicator in an analytics table.

## When not to use

- For raw values without a comparison reference — use plain text or a Badge.
- For statuses ("active", "draft") — use [Badge](?path=/docs/general-badge--docs).
- For complex chart-style trends — use a sparkline or a real chart.

## Anatomy

<Canvas of={StatTrendStories.Default} />

A `<span>` with `role="status"` and an `aria-label` describing the trend. The icon is decorative.

## Directions

<Canvas of={StatTrendStories.Directions} />

`up` renders an up arrow with success colour; `down` an arrow down with destructive; `flat` a minus with neutral.

```vue
<StatTrend :value="9.7" direction="up" severity="warning" />
```

`severity` is ignored when `direction === 'flat'` — the indicator always renders a `Minus` for no change.

## Hover

<Canvas of={StatTrendStories.WithHover} />

For richer context (comparison label, sparkline, link to a report), pass `hover` and fill the `#hover` slot with whatever you need. The wrapper is a `HoverCard` from `@fromforgesoftware/vue-kit`, so it already handles delays, focus, and dismissal.

```vue
<StatTrend :value="9.7" direction="up" severity="warning" hover>
  <template #hover>
    <p class="text-xs font-medium">vs same week last month</p>
    <p class="text-xs text-muted-foreground">+9.7% (vs 184)</p>
    <Sparkline :data="trendValues" />
  </template>
</StatTrend>
```

`open-delay` (200 ms) and `close-delay` (100 ms) defaults are tuned for inline-table use; override per call site if needed.

## Inverted sentiment

<Canvas of={StatTrendStories.InvertedSentiment} />

For metrics where **lower is better** (errors, latency, churn), pass `invert-sentiment` to flip the colour mapping. The icon stays driven by the actual direction; only the colour reverses.

```vue
<StatTrend :value="3.2" direction="down" invert-sentiment />
```

## Accessibility (WCAG 2.2 AA)

- **1.1.1 Non-text Content**: the arrow icon has `aria-hidden="true"`. The trend is announced via the wrapper's `aria-label`.
- **1.3.1 Info and Relationships**: rendered as `<span role="status">` so the trend is not a heading or list item.
- **1.4.3 Contrast**: every tone × sentiment combination meets 4.5 : 1 against its background.
- **4.1.3 Status Messages**: `role="status"` triggers a polite live-region announcement.

## Responsive behaviour

<Canvas of={StatTrendStories.InteractiveResponsive} />

StatTrend is `inline-flex` with `whitespace-nowrap` — it never wraps. Wrap multiple trends in a `flex flex-wrap` container if needed.
