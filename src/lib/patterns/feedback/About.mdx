# FeedbackPopover

A popover form that collects user feedback text. The trigger button opens a popover containing a heading, a textarea, and a submit button. After successful submission the parent calls `setSuccess()` which briefly shows a thank-you state before automatically closing.

## When to use

- Collecting in-product feedback, bug reports, or feature requests without leaving the current page.
- Any lightweight form that fits in a small popover rather than a full dialog.
