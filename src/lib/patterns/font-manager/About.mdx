# FontManager

A font size selector that adjusts the global text scale of the application. The selected size is persisted to `localStorage` and applied to the root `<html>` element so the entire UI scales proportionally.

## When to use

- In appearance/accessibility settings panels.
- When your application supports user-adjustable text density.
