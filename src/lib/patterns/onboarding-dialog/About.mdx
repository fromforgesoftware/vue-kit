# OnboardingDialog

A guided multi-step dialog for onboarding flows. Each step can include a media area (image, illustration, or custom content), a title, a description, and navigation via Next/Skip/Finish buttons. Step progress is shown via dot indicators.

## When to use

- When users first sign up or activate a feature, to walk them through key concepts step by step.
- For product tours triggered manually by the user (e.g., a "Show me how" button).
