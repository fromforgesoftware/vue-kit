# PlanPicker

A dialog for selecting and changing subscription plans. Displays plan options as selectable radio cards,
an optional features checklist, and confirm/cancel actions.

## When to use

- When a user needs to upgrade, downgrade, or switch their subscription tier.
- When you want to present plan options in a focused, modal context without navigating away.
