# Sort

A popover-based multi-field sort control. Users can add multiple sort fields, pick a field from a select, toggle ASC/DESC direction, delete fields, and drag rows to reorder them. Closing the popover commits the sort state and emits `update:modelValue`.

## When to use

- Any list, table, or grid view where users need to sort by one or more fields.
- Prefer Sort over inline column headers when the sort configuration is complex (multiple fields, custom direction labels).
