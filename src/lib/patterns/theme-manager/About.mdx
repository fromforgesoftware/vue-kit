# ThemeManager

A theme selector that lets users switch between preset themes (Light, Dark, Auto) and optionally configure a fully custom color palette. Preferences are persisted to `localStorage`.

## When to use

- In appearance/settings panels where users can control the application theme.
- When your design system supports multiple color themes including a user-defined custom theme.
