# Tour

A guided walkthrough that highlights one UI element at a time with a spotlight overlay and a popover card describing the step. Built on top of Reka UI Popover for placement and a custom SVG mask for the spotlight cutout.

## When to use

- First-run onboarding for net-new users.
- Re-introducing a feature after a redesign.
- Self-service product tours triggered from a "Take the tour" link in help menus.

## When not to use

- For required workflows (use a wizard / multi-step form instead).
- For one-off hints — use a [Tooltip](?path=/docs/general-tooltip--docs) or inline help text.
- For long-form documentation — link to a help-centre article.

## Anatomy

<Canvas of={TourStories.Default} />

A tour consists of an array of `TourStep` objects. Each step targets a CSS selector on the page, a placement for the popover, and a title/description.

## Auto-start

Pass `autoStart` to begin the tour as soon as the component mounts — typical for a first-run experience.

<Canvas of={TourStories.AutoStart} />

## Placement

Each step's `placement` controls which side the card appears on. Reka UI's collision detection flips the placement automatically when there is no room.
