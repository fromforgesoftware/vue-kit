# UnsavedChanges

Two complementary components for protecting users from accidentally losing unsaved work:

- **UnsavedChangesDialog** — a modal alert dialog that blocks navigation with Leave/Stay actions.
- **UnsavedChangesBanner** — a floating bottom bar with Save/Discard actions, optionally animated with a jiggle on blocked navigation.

## When to use

- Show the **Dialog** when the user tries to navigate away (route change, browser tab close) while a form has pending changes.
- Show the **Banner** as a persistent inline cue that unsaved changes exist, giving the user easy access to save or discard.
